## üèÅ Getting Started

### 1. Installation

```bash
npm install
```

### 2. Run Locally

```bash
npm run dev
```

### 3. Environment Variables

Create a `.env.local` file with the following keys:

```env
NEXT_PUBLIC_POSTHOG_KEY=your_key
NEXT_PUBLIC_POSTHOG_HOST=your_host
```
